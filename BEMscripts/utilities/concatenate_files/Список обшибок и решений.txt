1) Если в windows при запуске скриптов с библиотекой ffmpeg выходит ошибка :
  Error: Cannot find ffmpeg
    at C:\Users\Семенов\Documents\prokek\BEMscript\node_modules\fluent-ffmpeg\lib\processor.js:136:22
  Значит нужно добавить в переменные среды путь к испольняющему файлу ffmpeg
  который можно скачать с ресурса https://ffmpeg.org/ (тебе нужны exe файлы)

  Далее следуй инструкции:
  1. Открой свойства системы win + pause
  2. Найди  "Дополнительные параметры системы".
  3. Найди "Переменные среды".
  4. В разделе "Системные переменные" найдите переменную "Path" и нажмите на кнопку "Изменить".
  5. Добавьте путь к исполняемому файлу ffmpeg (например, "C:\путь\к\ffmpeg") в список значений переменной Path, разделяя его точкой с запятой от других значений.
  6. Нажмите "ОК" во всех открытых окнах, чтобы сохранить изменения.

  В macOS и Linux:
1. Откройте терминал.
2. Выполните команду sudo nano /etc/paths, чтобы открыть файл с переменными среды PATH для редактирования.
3. Добавьте путь к исполняемому файлу ffmpeg (например, "/путь/к/ffmpeg") на новой строке.
4. Нажмите Ctrl+X, затем Y, затем Enter, чтобы сохранить изменения и закрыть файл.
